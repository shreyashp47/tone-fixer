Tone Fixer — API Key Guide

Tone Fixer needs an API key to work. An API key is like a password that lets the extension talk to the AI. Pick a provider below, follow the steps, and copy-paste the key into Settings. Groq is free — start there.

Groq Free

Groq offers a free tier with high-speed inference. No credit card required.

  1. Go to console.groq.com/keys
  2. Sign in with Google, GitHub, or email
  3. Click "Create API Key"
  4. Give it a name (e.g. "Tone Fixer") and click "Create"
  5. Copy the key (it starts with gsk_)
  6. Paste it in Tone Fixer Settings and click Save
Recommended: Groq is completely free with generous rate limits. Perfect for getting started.

Google Gemini Billing

Gemini has a free tier with some limits; billing required for higher usage.

  1. Go to aistudio.google.com/apikey
  2. Sign in with your Google account
  3. Click "Create API Key"
  4. Copy the key (it starts with AIza)
  5. Paste it in Tone Fixer Settings and click Save

ChatGPT (OpenAI) Billing

OpenAI requires adding billing info even for low usage.

  1. Go to platform.openai.com/api-keys
  2. Sign in (create an account if needed)
  3. Click "+ Create new secret key"
  4. Give it a name and click "Create secret key"
  5. Copy the key (it starts with sk-proj-)
  6. Paste it in Tone Fixer Settings and click Save

Claude (Anthropic) Billing

Anthropic requires billing. They offer some free credits for new accounts.

  1. Go to console.anthropic.com/settings/keys
  2. Sign in (create an account if needed)
  3. Click "Create Key"
  4. Copy the key (it starts with sk-ant-)
  5. Paste it in Tone Fixer Settings and click Save

xAI Grok Billing

xAI requires a paid account for API access.

  1. Go to console.x.ai
  2. Sign in with your X (Twitter) account
  3. Set up billing in the console
  4. Create an API key from the dashboard
  5. Copy the key (it starts with xai-)
  6. Paste it in Tone Fixer Settings and click Save
